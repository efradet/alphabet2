# Alphabet Voice Game (React + Vite + TypeScript + Redux + Tailwind)
- 30s par lettre. Barre **verte 0–20s**, **rouge 25–30s**.
- Transcription **en direct** affichée.
- Tolérance **fuzzy** aux petites erreurs (bé/bee, doubleve/double vé, etc.).
- Le jeu s'arrête quand toutes les lettres sont reconnues.
## Lancer
npm install
npm run dev
## Conseil
Chrome desktop recommandé. Cliquez **Commencer** pour autoriser le micro.
